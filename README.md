# 待办清单

A simple todo application with dark/light theme support.

## Features

- Add, edit, and delete todos
- Mark as complete/incomplete
- Priority levels (high/medium/low) - click badge to cycle
- Dark/light theme toggle
- Local data persistence (localStorage)

## Usage

Open `index.html` in your browser.

## Tech Stack

- HTML / CSS / JavaScript
- No dependencies, no build tools required

# 待办清单

一个简洁的待办事项应用，支持深色/浅色主题切换。

## 功能

- 添加、编辑、删除待办事项
- 标记完成/未完成
- 优先级设置（高/中/低）- 点击徽章可切换
- 深色/浅色主题切换
- 数据本地持久化（localStorage）

## 使用方法

直接在浏览器中打开 `index.html` 即可使用。

## 技术栈

- HTML / CSS / JavaScript
- 无依赖，无需构建工具
